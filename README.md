## Telegram ios 解锁

1.  升级 Telegram ios最新版
2.  升级 Telegram Desktop最新版
3.  打开 Telegram Desktop
4.  点击 `Settings | 设置`
5.  点击 `Privacy and Security | 隐私与安全`
6.  点击 `Sensitive content | 敏感内容`
7.  打开 `Disable filtering | 停止过滤`
8.  重启 iOS客户端
